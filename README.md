# pptx-md

Convert PPTX files into LLM-friendly Markdown (VLM-first image understanding).

> **Work in progress** — Full documentation and usage examples will be added in M6 (FR-18).

## Installation

```bash
pip install pptx-md
```

With VLM support:

```bash
pip install pptx-md[vlm]
```

## Requirements

- Python 3.11+
